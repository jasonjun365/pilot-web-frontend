## Install

1. yarn install

## Run

1. yarn start:dev
2. yarn start:uat
3. yarn start:prod

## Build

1. yarn build:dev
2. yarn build:uat
3. yarn build:prod

## Env(.dev, .pre, .prod)

1. CUSTOM_ENV (Env)
2. CUSTOM_BASENAME (History router basename)
3. CUSTOM_BASE_URL (Base Url)
4. CUSTOM_MEDIA_UPLOAD (Upload Image Url)
6. CUSTOM_MEDIA_READ (Read Image url)

## Common Env(.common)

1. CUSTOM_PROJECT_NAME (Project Name)
